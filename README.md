🎮 Tic-Tac-Toe AI

Search & Machine Learning Project

📌 Overview

This project is an AI-powered Tic-Tac-Toe game that combines classical search algorithms with machine learning techniques to create intelligent decision-making.

The system uses Minimax with Alpha-Beta Pruning for optimal move selection and a Perceptron-based evaluation function to assess board states.

🚀 Features
🎯 AI opponent with intelligent move selection
⚡ Minimax algorithm with Alpha-Beta pruning optimization
🧠 Machine Learning evaluation using Perceptron
🔄 Hybrid AI approach (Search + ML)
🎮 Multiple difficulty levels (Easy, Normal, Hard)
🖥️ Interactive GUI built with Java Swing
🧠 AI Approach
🔹 Search Algorithm
Minimax algorithm explores possible game states
Alpha-Beta pruning reduces unnecessary computations
Ensures optimal or near-optimal decisions
🔹 Machine Learning
Perceptron model evaluates board positions
Uses extracted features from game states
Provides heuristic values instead of fixed rules
🔹 Hybrid System
ML model evaluates states
Minimax uses these evaluations to choose the best move
🛠️ Technologies Used
Java
Java Swing (GUI)
Machine Learning (Perceptron)
Search Algorithms (Minimax, Alpha-Beta Pruning)
